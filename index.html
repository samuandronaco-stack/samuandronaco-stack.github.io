TikTok Content Posting API

Automate your TikTok
carousel publishing

A content management tool for creators managing multiple TikTok accounts across different markets. Upload photo carousels from Google Drive to TikTok in seconds.

📸

Google Drive Integration

Organize your carousel images in Drive folders. The tool reads them automatically in the correct order and downloads them for processing.

▶

TikTok Direct Post

Uses TikTok Content Posting API to upload photo carousels directly to authorized creator accounts as private drafts, ready for review and publishing.

🌐

Multi-Market Support

Manage 5 language markets (IT, DE, ES, FR, US) with separate TikTok accounts, Drive folders, and localized titles — all from one tool.

📅

Monthly Title Scheduling

Import a monthly titles file and the tool automatically selects the right title for each carousel based on the upload date and target market.

How it works

1

Authenticate

Connect each TikTok creator account securely via OAuth 2.0 with PKCE.

2

Select carousel

Choose the Drive folder containing the carousel images for the target market.

3

Upload

Images are processed, temporarily hosted, and posted to TikTok via Content Posting API.

4

Publish

The creator reviews the draft on TikTok, adds music, and publishes it to their audience.
